Research

Isotropic Neural Cellular Automata

Neural Cellular Automata (NCA) are an extension of cellular automata that have the ability to learn emergent morphogenetic behavior using convolutional neural networks. This project involves modifying the original model to include a Laplacian perception filter and testing the out-of-training isotropic and inferential properties of this new model using PyTorch.

Programmable Self-Organizing Robots

Reconfigurable robots can approach tasks like mapping, search-and-rescue, and accessibility far more efficiently than conventional robots. However, there is still some limitation in the algorithms presently used to drive self-organization. The focus of this work is to use cellular automata as a framework for developing collaborative algorithms using graph theory and neural networks.

Math

Bertrand's Postulate

Bertrand's Postulate was first described as a conjecture in 1845 by Joseph Bertrand, who later proved it for all numbers between 3 and 3,000,000. Because the statement was completely proved by P. Chebyschev in 1852, the postulate is sometimes referred to as the "Bertrand-Chebyshev Theorem" or "Chebyshev's Theorem." The theorem states that for any integer x greater than 3, there exists at least one prime number between x and 2x-2.

Collatz's Conjecture

Also known as "Hasse's Algorithm" and the "3n+1 Conjecture," the Collatz Conjecture observes an assumption regarding sequences of known positive integers. The conjecture states that if the current integer is even, the term that follows it is half the current term. Otherwise, the following term is the current term multiplied by 3 then added to 1. By these rules, the sequence is stated to eventually terminate at 1, regardless of the initial integer.

Goldbach's Conjecture

One of the oldest unsolved problems in number theory, Goldbach's conjecture was first proposed in a letter to Leonard Euler in 1742. Christian Goldbach's idea states that if an integer is even and greater than 2, it can be expressed as the sum of at least one set of two primes. For example, 6 can be written as 3+3 and 8 can be expressed as 3+5. This conjecture has been proved for the natural numbers up to 4E18, though it has yet to be completely solved.

Wilson's Theorem

Proved by Joseph-Louis Langrange in 1770, Wilson's Theorem was proposed by John Wilson in 18th century to determine whether or not a positive integer is a prime number. The theorem states that if the factorial of a number minus 1 in the modulo of that number equals -1 in the modulo of that number, then the number is prime. For example, the number 2, a known prime number, statisfies this theorem because the factorial of 2-1 is equal to -1 mod 2.
